# checkpoint jsx

Dépôt créé depuis le workspace local par Copilot.

Ce projet contient une petite application React affichant une carte produit.

Commandes:

- Installer les dépendances: `npm install`
- Lancer le serveur: `npm start`

Astuce: modifiez la variable `prenom` dans `src/App.js` pour changer le message de salutation.
